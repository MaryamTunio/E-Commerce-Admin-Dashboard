#  E-Commerce Admin Dashboard

A modern, responsive **E-Commerce Admin Dashboard** built with **HTML, CSS, JavaScript, Bootstrap 5, and Chart.js**. The dashboard provides key business insights through interactive charts, KPI cards, and a clean Glassmorphism-inspired user interface.

---

##  Features

* Interactive Dashboard Overview
*  KPI Cards (Revenue, Orders, Customers, Conversion Rate)
* Weekly Sales Line Chart
* Top Selling Products Bar Chart
* Traffic Sources Doughnut Chart
* Date Filter Simulation (Today, Last 7 Days, Last 30 Days, This Year)
*  Refresh Dashboard Button
*  User Profile Management
*  Client-side Form Validation
*  Loading Screen Animation
*  Fully Responsive Design
*  Modern Glassmorphism UI

---
